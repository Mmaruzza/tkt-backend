
# Ticketera Trans

Nueva version de la Ticketera Trans en React y Node.js

## Primeros pasos
  Front-end: 
```bash
  cd ticketera-website
  npm install
  npm run dw
```

  Back-end: 
```bash
  cd TICKETERA-BACKEND
  npm start
```

  Api gateway: 
```bash
  cd API-GATEWAY
  npm start
```
